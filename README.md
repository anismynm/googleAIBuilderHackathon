# 📺 SceneExplore — TV 4방향키 탐험 서비스

**Powered by Google Genie 3 × RealWonder**

---

## 1. 서비스 요약

**SceneExplore**는 시청자가 TV를 보던 중 원하는 장면에서 리모컨의 4방향키(↑↓←→)를 누르면, 그 장면 속 세계를 자유롭게 탐험할 수 있는 인터랙티브 AI 서비스입니다. 

예컨대 드라마에서 주인공이 해변을 걷는 장면이 나올 때, 방향키로 카메라를 왼쪽으로 돌리면 — 원본 영상에는 존재하지 않는 해변 너머의 공간이 AI에 의해 실시간으로 생성되어 화면에 펼쳐집니다. 단순한 '일시정지'를 넘어, 정지된 프레임 한 장에서 새로운 3D 세계로 진입하는 몰입형 경험을 제공합니다.

기술 스택은 두 축으로 구성됩니다:
*   **Google Genie 3**: 사용자가 이동하면 실시간으로 주변 세계를 생성하며, 720p 해상도에서 초당 20~24프레임의 유동적 인터랙션을 제공하는 오토-리그레시브 월드 모델로, 장면의 공간적 확장을 담당합니다.
*   **RealWonder**: 단일 이미지에서 3D 재구성, 물리 시뮬레이션, 4 디퓨전 스텝의 경량 영상 생성기를 통합하여 13.2 FPS의 실시간 물리 기반 인터랙션을 가능하게 하는 모델로, 장면 내 오브젝트와의 물리적 상호작용을 담당합니다.

## 2. 핵심 기능

### ① 탐험 모드 진입 (Scene Entry)
TV 시청 중 4방향키를 누르면 영상이 일시정지되고 탐험 모드로 전환됩니다. 현재 시청 중이던 프레임이 탐험의 출발 세계가 됩니다.

### ② 실시간 시점 이동 (Directional World Navigation)
방향키 입력에 따라 카메라가 이동하며, 사용자가 이동하고 카메라를 조정할 때마다 Genie 3가 다음 장면을 실시간으로 예측·생성하여 AI가 만든 세계를 탐험하는 완전히 새로운 방식을 제공합니다. 원본 영상에 없던 공간이 자연스럽게 펼쳐집니다.

### ③ 물리 기반 오브젝트 인터랙션 (Physics Interaction)
RealWonder는 강체, 변형체, 유체, 입자 물질 등에 대해 힘, 로봇 동작, 카메라 컨트롤 등 다양한 물리적 액션을 통한 인터랙티브 탐험을 지원합니다. 장면 속 오브젝트를 선택하면 물리 법칙에 따라 반응하는 정교한 상호작용이 가능합니다.

### ④ 컨텍스트 일관성 유지 (Context Coherence)
이전에 탐험한 세부 내용은 재방문 시 기억되며, 지속적인 상호작용에도 환경 퀄리티가 저하되지 않습니다. 원본 콘텐츠의 시대, 장르, 조명, 분위기가 새롭게 생성된 공간 전반에 걸쳐 완벽하게 일관되게 유지됩니다.

### ⑤ 탐험 종료 및 복귀 (Return to Content)
확인(OK) 버튼을 누르면 탐험 모드가 즉시 종료되고, 시청하던 장면으로 자연스럽게 복귀하여 중단 없이 콘텐츠 감상을 이어갈 수 있습니다.

## 3. 혁신성 (기업 관점)

*   **패러다임 전환 — 수동적 시청에서 능동적 탐험으로**
    TV는 지난 수십 년간 시청자가 제작자가 설계한 시점과 흐름을 일방적으로 수용하는 선형 매체였습니다. SceneExplore는 기업이 이러한 기존 구조를 처음으로 깨고 능동형 시청 경험을 선점할 수 있는 서비스입니다. 기존 Veo, Sora 같은 AI 영상 모델들이 사전에 결정된 영상을 재생하는 데 그치는 반면, Genie 3는 사용자의 행동에 맞춰 새로운 프레임을 실시간으로 예측하고 생성하는 근본적으로 다른 접근을 취합니다.
*   **기술 우위의 희소성**
    인터랙티브하고 탐험 가능한 환경 구축은 단순 영상 생성과는 차원이 다른 기술적 도약입니다. 현재 이 영역에서 선도적인 위치에 있는 Google DeepMind의 기술을 TV 플랫폼에 최초 접목함으로써 강력한 시장 선점 효과를 누릴 수 있습니다.
*   **산업 파괴력의 입증**
    인터랙티브 세계 생성 AI의 등장은 기존 콘텐츠 산업 구조를 뒤흔들 잠재력을 보여주었습니다(Project Genie 공개 직후 관련 게임/엔진 주가 변동 사례 등). 이는 TV 플랫폼 기업에게 새로운 기회이자, 경쟁사 대비 선제 도입이 필수적인 당위성을 제공합니다.

## 4. 비즈니스 영향도

### B2B 비즈니스 모델
*   **OTT·방송사 대상 SDK 라이선싱**: SceneExplore를 SDK 형태로 패키징하여 Netflix, Disney+, 웨이브 등 주요 OTT 플랫폼에 공급합니다. 탐험 기능은 포화된 시장에서의 강력한 차별화 무기이며, 프리미엄 구독 기능으로 ARPU(가입자당 평균 수익) 향상에 직결됩니다.
*   **TV 제조사 파트너십**: 삼성, LG 등 프리미엄 TV 라인업의 킬러 AI 기능으로 탑재합니다. 하드웨어의 프리미엄 가치를 상승시키고 신규 수익 채널을 창출할 수 있습니다.
*   **광고 플랫폼 신규 포맷 (Contextual Exploration Ad)**: 탐험 중 등장하는 공간에 브랜드 스폰서 장면을 자연스럽게 생성하는 맞춤형 AI 광고 상품입니다. 기존 PPL보다 훨씬 몰입도 높은 브랜드 경험을 제공합니다.
*   **콘텐츠 제작사 협력**: 제작 단계에서 탐험용 공간 메타데이터를 설계하는 새로운 표준을 수립하고 솔루션화하여 B2B 파이프라인으로 확장합니다.

### B2C 비즈니스 모델
*   **프리미엄 구독 수익화**: 탐험 기능을 상위 구독 티어에 배치하여 신규 유료 전환 및 업셀링을 유도합니다.
*   **T-Commerce 2.0 (즉시 구매 연결)**: 드라마 속 배경, 인테리어, 소품을 탐험하며 해당 상품을 즉시 구매하는 흐름을 구축합니다. 구매 맥락이 자연스러워 전환율이 극대화됩니다.
*   **팬덤 및 IP 확장**: 세계관이 탄탄한 판타지, SF 장르 등에서 팬들이 작품 속 세계를 직접 탐험하게 함으로써 IP 몰입도와 충성도를 폭발적으로 높일 수 있습니다.

> Google Genie 3의 실시간 세계 생성 능력과 RealWonder의 물리 기반 인터랙션이 결합될 때, 비로소 화면 밖의 세계를 탐험하는 혁신적인 경험이 실현됩니다.

## 🛠 실제 구현 기술 스택

본 프로젝트의 동작 프로토타입은 다음 기술들을 활용하여 구현되었습니다.

- **Frontend:** React (TypeScript), CSS
- **Backend:** Node.js (Express)
- **AI / API:** Google Cloud Vertex AI API (탐험 중 영상 내 오브젝트 자동 인식 및 메타데이터 추출용)

## 🚀 실행 방법 (How to Run)

이 프로젝트는 프론트엔드(React)와 백엔드(Node.js) 서버를 각각 실행해야 합니다.

### 1. 백엔드 실행 (Backend)

백엔드 서버는 영상 정보 처리 및 Vertex AI API와의 통신을 담당합니다.

```bash
# 백엔드 디렉토리로 이동
cd click-to-explore/backend

# 패키지 설치
npm install

# 환경변수 설정
# .env.local 파일을 생성하고 필요한 설정(예: GCP_PROJECT_ID, GOOGLE_APPLICATION_CREDENTIALS 등)을 입력하세요.

# 백엔드 서버 실행 (일반적으로 http://localhost:8080 등에서 실행됨)
npm start
```

### 2. 프론트엔드 실행 (Frontend)

프론트엔드는 사용자에게 보여지는 TV 화면입니다.

```bash
# 프로젝트 루트(click-to-explore) 디렉토리에서 실행 (또는 새 터미널 창 열기)
cd click-to-explore

# 패키지 설치
npm install

# 리액트 개발 서버 실행
npm start
```
> 프론트엔드 서버가 실행되면 브라우저에서 자동으로 `http://localhost:3000` 이 열리며 앱을 확인할 수 있습니다.
